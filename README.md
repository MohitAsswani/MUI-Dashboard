Evaluation Project: B2B Data Labelling Platform Prototype

Objective for the Project
Create a simplified version of a dashboard that should have the following implementation :
- Implement a list view of data entries with the ability to select and edit individual fields.
- Provide a detailed view for editing selected data entries with an option to save
changes or mark for review.

Installation Instructions
To install the necessary dependencies, follow the steps below:

1. can clone the project
2. After cloning the project, run npm install --legacy-peer-deps. This will install all the necessary dependencies.
3. After the dependencies have been installed, run npm start. This will initialize the project and will start the localhost server of the project.
